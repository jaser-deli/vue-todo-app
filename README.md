## ✨ Features

- **📌 Task Management**
  - Add, edit, and delete todos with ease
  - Double-click to edit task text
  - One-click deletion
  - Mark tasks as complete

- **📅 Due Dates**
  - Set and modify due dates for any task
  - Smart date formatting (Today, Tomorrow, etc.)
  - Visual indicators for overdue tasks
  - Automatic sorting based on due dates

- **💾 Data Persistence**
  - Automatic saving to localStorage
  - Tasks persist across browser sessions
  - No backend required

- **🎨 Modern UI/UX**
  - Clean, intuitive interface
  - Responsive design for all devices
  - Smooth animations and transitions
  - Clear visual feedback

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- No additional installations required!

### Quick Start
1. Clone the repository:
   ```bash
   git clone https://github.com/jaser-deli/vue-todo-app.git
   ```

2. Navigate to the project directory:
   ```bash
   cd vue-todo-app
   ```

3. Open `index.html` in your browser:
   - Double-click the file in your file explorer, or
   - Drag and drop it into your browser, or
   - Use a local server:
     ```bash
     python -m http.server 8000
     ```
     Then visit `http://localhost:8000`

## 💻 Usage

1. **Adding Tasks**
   - Type your task in the input field
   - Press Enter or click "Add Task"

2. **Managing Tasks**
   - Click the checkbox to mark as complete
   - Double-click the text to edit
   - Click the calendar icon to set a due date
   - Click the trash icon to delete

3. **Organization**
   - Tasks are automatically sorted by:
     1. Completion status (incomplete first)
     2. Due date (if set)
     3. Creation time
